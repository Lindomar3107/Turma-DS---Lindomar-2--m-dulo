Edição de imagens com o GIMP

Resolução e Proporção

Uma imagem de fundo precisa ter uma resolução alta para não ficar pixelizada em telas maiores. A recomendação padrão é usar uma imagem com pelo menos 1920 pixels de largura por 1080 pixels de altura (proporção 16:9), que é a resolução Full HD, a mais comum em monitores e laptops hoje em dia. Se você quer garantir que a imagem fique bem até em monitores 4K, o ideal é usar uma resolução de 3840 x 2160 pixels. Também é crucial que a imagem tenha uma boa proporção. Uma imagem na proporção 16:9 se adapta bem à maioria das telas, mas se o design do seu site permite, você pode usar uma imagem mais ampla, como 21:9, para dar um toque mais cinematográfico. Tamanho do Arquivo Além da resolução, o tamanho do arquivo da imagem é extremamente importante. Uma imagem grande demais pode deixar o seu site lento, o que prejudica a experiência do usuário e o seu SEO. O ideal é que o arquivo tenha no máximo 500 KB.
